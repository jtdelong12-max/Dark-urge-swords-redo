# Dark-urge-swords-redo

A Baldur's Gate 3 mod featuring powerful demonic weapons with unique abilities.

## Recent Optimizations (Jan 2026)

This mod has been optimized for better performance and maintainability:

- **14.7% smaller file size** (108 lines reduced)
- **Faster loading times** through code consolidation
- **Easier maintenance** using inheritance patterns
- **Zero duplicate code** - eliminated redundant definitions

### What Changed?

Weapon definitions have been restructured using base templates, making the mod more efficient and easier to maintain. **All weapons function identically** - only the code structure improved.
